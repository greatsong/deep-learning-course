    5차시: 퍼셉트론과 신경망 — 선 하나로 안 되면?¶

    1. 수업 개요¶

    학습 목표¶

    • 선형 분류의 한계를 직접 체험하고, 직선 하나로 나눌 수 없는 문제가 있음을 안다.
    • 다층 신경망(Multi-layer Network)의 필요성을 XOR 문제를 통해 이해한다.
    • TensorFlow Playground에서 뉴런과 레이어를 조작하며 신경망의 구조를 탐구한다.

    준비물¶

    항목 수량 비고
    A4 프린트 (점 찍힌 종이) 1인 2장 쉬운 패턴 1장 + XOR 패턴 1장
    자 1인 1개 직선 긋기용
    노트북 모둠당 1대 이상 playground.tensorflow.org
    프로젝터/대형 모니터 1대 교사 시연 및 대결 결과 공유

    시간 배분¶

    단계 시간 활동 내용
    도입 7분 A4 직선 분류 활동 + XOR 충격
    이론 전개 10분 퍼셉트론 개념, AI 겨울 이야기, 다층 신경망
    실습 23분 TF Playground 대결
    정리 10분 핵심 정리 + 다음 차시 예고

    2. 도입 시나리오 (7분)¶

    교사 대사 전문¶

    "지난 시간에 Teachable Machine으로 AI를 만들었죠? 오늘은 그 AI 안에 뭐가 들어있는지 뜯어봅시다. AI의 가장 작은 부품부터 시작할 거예요."

    [A4 프린트 — 쉬운 패턴 배부]

    "자, 종이에 빨간 점과 파란 점이 찍혀 있죠? 미션은 간단합니다. 자로 직선을 딱 하나만 그어서 빨간 점과 파란 점을 나눠보세요. 30초 드릴게요. 시작!"

    [30초 후]

    "다 됐죠? 쉽죠? 이게 바로 AI의 가장 기본 단위가 하는 일이에요. 직선 하나로 '이쪽은 빨강, 저쪽은 파랑'을 나누는 거예요."

    [A4 프린트 — XOR 패턴 배부]

    "그럼 이건요? 두 번째 종이를 보세요. 역시 직선 하나로 나눠보세요. 30초!"

    [30초 후 — 학생들 웅성웅성]

    "안 되죠? 아무리 해도 직선 하나로는 안 되죠? 바로 이거예요. 1969년에 이 문제를 발견한 과학자가 있었는데, 그때 AI 연구가 거의 멈췄어요. 무려 10년 넘게. 이걸 'AI의 겨울'이라고 해요."

    XOR 프린트 준비 팁

    2x2 격자에 대각선으로 같은 색을 배치하세요: 좌상-빨강, 우상-파랑, 좌하-파랑, 우하-빨강. 점의 크기를 크게(지름 1cm 이상) 하고 여러 개씩 군집으로 찍으면 더 명확합니다.

    3. 이론 전개 포인트 (10분)¶

    설명 순서¶

    1단계: 퍼셉트론 = 직선 하나 (3분)

    "아까 직선 하나로 나눈 것, 그게 '퍼셉트론'이에요. 1957년에 만들어진 AI의 가장 기본 단위예요. 입력을 받아서 직선 하나로 '예/아니오'를 판단하는 장치입니다."

    [칠판에 간단한 퍼셉트론 다이어그램]

    입력1 ──┐
        ├──→ [ 뉴런 ] ──→ 출력 (빨강 or 파랑)
입력2 ──┘

    "입력이 들어오면, 가중치를 곱하고, 더해서, 기준을 넘으면 '빨강', 안 넘으면 '파랑'. 이게 전부예요."

    2단계: XOR 문제와 AI의 겨울 (3분)

    "그런데 아까 두 번째 종이... 직선 하나로 안 됐죠? 이걸 XOR 문제라고 해요. 1969년에 민스키라는 유명한 과학자가 '퍼셉트론은 이 문제를 못 푼다'고 증명했어요. 그러자 사람들이 'AI 연구해 봐야 소용없네' 하고 연구비를 끊었어요. 10년 넘게."

    역사 포인트

    AI의 겨울 이야기는 학생들에게 "과학도 실패와 좌절이 있다"는 메시지를 전달하는 좋은 기회입니다. 간략하게만 짚되, 감정적으로 전달하면 기억에 남습니다.

    3단계: 해결책 = 층을 쌓자! (4분)

    "그런데 누군가 이런 생각을 했어요. '직선 하나로 안 되면... 여러 개 쌓으면 되지 않을까?'"

    [칠판에 2층 신경망 그림]

    입력 ──→ [뉴런][뉴런] ──→ [뉴런] ──→ 출력
         (은닉층)         (출력층)

    "첫 번째 층에서 직선 2개를 긋고, 두 번째 층에서 그 결과를 합치면? XOR도 풀 수 있어요! 직선을 조합하면 곡선도, 복잡한 경계도 만들 수 있거든요. 이게 다층 신경망, 영어로 Multi-layer Network예요."

    "Teachable Machine도 안에 이런 층이 쌓여 있었던 거예요. 우리가 사진을 넣으면 여러 층의 뉴런들이 패턴을 찾아낸 거죠."

    4. 실습 운영 상세 — TF Playground 대결 (23분)¶

    사전 세팅 안내 (3분)¶

    [화면: playground.tensorflow.org]

    "자, 이 사이트가 TensorFlow Playground예요. 신경망을 직접 만져볼 수 있는 놀이터입니다."

    교사가 화면을 보여주며 인터페이스 설명:

    영역 설명 학생 안내 멘트
    왼쪽 데이터 점들의 패턴 "이게 우리가 나눠야 할 데이터예요"
    가운데 뉴런 신경망 구조 "여기가 AI의 뇌예요. 뉴런을 추가/삭제할 수 있어요"
    오른쪽 출력 분류 결과 "AI가 어떻게 나눴는지 색으로 보여줘요"
    상단 재생 버튼 학습 시작/정지 "이걸 누르면 AI가 학습을 시작해요"
    Test loss 손실값 "이 숫자가 작을수록 잘 분류한 거예요"

    대결 규칙 안내 (2분)¶

    "모둠 대결을 할 거예요! 규칙을 알려줄게요."

    대결 규칙

    1. 데이터셋: spiral (나선형) — 왼쪽 아래 데이터 패널에서 선택
    2. 목표: Test loss를 0.1 이하로 만들기
    3. 승리 조건: 가장 적은 뉴런 수로 목표 달성한 모둠 우승
    4. 통일 설정: Learning rate = 0.03, Activation = ReLU
    5. 시간 제한: 10분

    대결 진행 (10분)¶

    교사 순회 가이드¶

    상황 교사 개입
    뉴런을 너무 많이 추가 "뉴런 20개면 되긴 하는데, 8개로도 가능해요. 줄여볼래요?"
    레이어 추가를 모름 "가운데 + 버튼으로 층을 추가할 수 있어요"
    Loss가 안 내려감 "Epoch 수를 확인하세요. 충분히 학습했나요? 혹시 뉴런이 너무 적진 않은지도요."
    바로 성공한 모둠 "축하해요! 이제 뉴런 수를 최소로 줄여보세요. 최소 기록에 도전!"

    교사용 정답 가이드

    spiral 데이터셋의 경우, 2개 은닉층 x 각 4뉴런 = 총 8뉴런 정도면 Loss 0.1 이하 달성 가능합니다. 3개 은닉층을 사용하면 더 적은 뉴런으로도 됩니다.

    결과 공유 (8분)¶

    "시간 끝! 각 모둠의 결과를 볼게요."

    공유 항목 방법
    뉴런 수 칠판에 모둠별 기록
    레이어 구조 "몇 층에 뉴런 몇 개?"
    Test loss 최종 수치
    시각화 프로젝터로 최우수 모둠 화면 공유

    "가장 적은 뉴런으로 성공한 모둠은 OO모둠, 총 O개! 뉴런이 적다는 건 효율적인 AI라는 뜻이에요."

    "뉴런 하나하나가 어떤 패턴을 잡는지 보이시나요? 가운데 뉴런 위에 마우스를 올려보세요. 각 뉴런이 담당하는 영역이 보여요. 이걸 합치면 나선형을 분류할 수 있게 되는 거예요."

    5. 학생 예상 반응 & 교사 피드백¶

    도입 단계 (A4 활동)¶

    학생 반응 교사 피드백
    "선을 구부리면 안 돼요?" "규칙이 직선 하나예요. 직선 하나가 퍼셉트론이거든요. 하나로 안 되면? 여러 개 쓰는 거죠!"
    "대각선으로 그으면 돼요!" (쉬운 패턴) "맞아요! 어떤 각도든 직선 하나면 되니까 퍼셉트론으로 풀 수 있는 문제예요."
    "두 번째는 진짜 안 되는데요?" "네, 안 됩니다. 수학적으로 증명된 거예요. 그래서 다른 방법이 필요한 거예요."

    이론 단계¶

    학생 반응 교사 피드백
    "이게 왜 중요해요?" "1969년에 이 문제 때문에 AI 연구가 거의 멈췄어요. 10년 넘게. 하지만 누군가 '층을 쌓자'는 아이디어를 냈고, 그게 지금의 딥러닝이 된 거예요."
    "뉴런이 진짜 뇌의 뉴런이에요?" "영감을 받았지만 실제 뇌와는 많이 달라요. 이름만 비슷한 거예요. 진짜 뇌는 훨씬 더 복잡하고 신비로워요."
    "레이어가 뭐예요?" "층이에요. 판단을 여러 단계로 나누는 거예요. 1층에서 대략 나누고, 2층에서 더 세밀하게 나누고... 이런 식이에요."
    "딥러닝의 '딥'이 이 층이에요?" "정확해요! 층을 깊이(deep) 쌓는다고 해서 딥러닝이에요. 아주 좋은 관찰이에요."
    (어려워하는 학생) "너무 어려워요" "핵심만 기억하세요: 선 하나로 안 되면 여러 개 쌓자! 이것만 알면 돼요."

    실습 단계 (TF Playground)¶

    학생 반응 교사 피드백
    "뭘 눌러야 해요?" "왼쪽 아래에서 spiral 데이터를 선택하고, 재생 버튼을 누르면 학습이 시작돼요. 가운데에서 + 버튼으로 뉴런을 추가할 수 있어요."
    "Loss가 0이 안 돼요" "0에 가까우면 충분해요. 0.05 이하면 아주 잘한 거예요. 완벽한 0은 오히려 과적합일 수 있어요."
    "다른 모둠보다 뉴런이 많은데..." "적은 뉴런이 목표지만, 일단 성공하는 게 중요해요. 성공한 후에 하나씩 줄여보세요."
    "ReLU가 뭐예요?" "뉴런이 신호를 처리하는 방식이에요. 음수는 0으로 만들고, 양수는 그대로 통과시키는 필터라고 생각하세요."
    "다른 데이터셋도 해보고 싶어요" "대결 끝나고 시간 남으면 해보세요! circle, exclusive or 등 다양한 패턴이 있어요."

    6. 수준별 분화 전략¶

    상위 수준 (코딩 관심 학생)¶

    심화 활동

    • 뉴런별 시각화 해석: 각 뉴런이 어떤 경계를 학습했는지 마우스 오버로 관찰
    • 활성화 함수 비교: ReLU vs Sigmoid vs Tanh로 바꿔보고 차이 관찰
    • 최소 뉴런 도전: spiral을 6개 이하 뉴런으로 풀어보기
    • 데이터셋 비교: circle, XOR, spiral 각각에 필요한 최소 구조 탐색

    중위 수준 (보통 학생)¶

    기본 목표

    • 대결에 적극 참여하여 Loss 0.1 이하 달성
    • 뉴런 추가/삭제, 레이어 추가의 효과를 관찰
    • "뉴런이 많을수록 복잡한 패턴을 나눌 수 있다"는 결론 도출

    하위 수준 (어려워하는 학생)¶

    지원 방안

    • 교사가 함께 화면을 보며 조작 안내
    • "뉴런 추가" 버튼만 직접 클릭하게 하고, 나머지는 교사가 설정
    • 시각적 변화에 집중: "색이 어떻게 바뀌었나요?" "점들이 잘 나뉘었나요?"
    • A4 활동에서의 체험을 자주 연결: "아까 직선으로 안 됐던 거, 여기서 여러 개 쓰니까 되죠?"

    7. 자주 나오는 질문 FAQ¶

    Q1. TensorFlow Playground가 접속이 안 돼요.

    학교 네트워크에서 차단될 수 있습니다. 전날 확인이 필수입니다. 대안으로 교사가 화면을 프로젝터로 공유하고, 학생들이 "3번 뉴런 추가해 주세요" 식으로 지시하는 방식도 가능합니다.

    Q2. 퍼셉트론과 뉴런의 차이가 뭔가요?

    퍼셉트론은 1957년 Frank Rosenblatt가 만든 초기 모델의 이름이고, 뉴런(노드)은 신경망의 구성 요소를 부르는 일반적인 용어입니다. 수업에서는 "퍼셉트론 = 가장 단순한 뉴런 하나"로 설명하면 됩니다.

    Q3. 학생이 미분이나 수학적 원리를 물어보면?

    "가중치를 조절해서 최적의 직선을 찾는 과정이 있는데, 그게 수학적으로는 미분을 사용해요. 하지만 오늘은 '직선 여러 개를 쌓는다'는 구조만 이해하면 충분해요. 수학은 대학에서 배울 수 있어요."

    Q4. AI의 겨울은 왜 끝났나요?

    1986년에 제프리 힌턴 등이 "역전파(Backpropagation)" 알고리즘을 발표하면서 다층 신경망을 효과적으로 학습시킬 수 있게 되었습니다. 이후 컴퓨터 성능 향상, 빅데이터와 맞물리며 지금의 딥러닝 시대가 열렸습니다.

    Q5. spiral 데이터 대결에서 적절한 힌트 수준은?

    처음 3분은 자유 탐색, 5분 시점에 "은닉층을 2개 이상 해보세요"라고 힌트, 8분 시점에 "각 층에 뉴런 4개씩 해보세요"라고 구체적 힌트를 줍니다. 단계적으로 힌트를 주면 성취감과 학습 효과를 동시에 얻을 수 있습니다.

    8. 정리 & 다음 차시 연결 멘트 (10분)¶

    오늘 배운 핵심 정리¶

    "오늘 뭘 배웠는지 세 줄로 정리할게요."

    번호 핵심
    1 퍼셉트론 = 직선 하나로 분류하는 가장 단순한 AI
    2 직선 하나로 안 되면(XOR 문제) → 층을 쌓아서 해결
    3 층이 깊어질수록 복잡한 패턴도 분류 가능 = 딥러닝

    "아까 A4에서 직선 하나로 안 됐던 게, Playground에서 뉴런 여러 개 쌓으니까 나선형도 분류됐죠? 바로 그거예요."
