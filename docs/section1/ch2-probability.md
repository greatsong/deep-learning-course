# 2차시: AI에게 창의적인 답을 시키려면? (확률과 Temperature)

## 이번에 배울 것

- [x] AI가 다음 단어를 어떻게 고르는지 이해해요
- [x] 확률 분포가 무엇인지 알아요
- [x] Temperature가 AI 답변에 어떤 영향을 주는지 실험해요
- [x] AI가 "생각"하는 것과 "확률적으로 고르는 것"의 차이를 설명할 수 있어요

---

## 도입: 같은 질문, 다른 답?

ChatGPT Playground에서 **Temperature 슬라이더**를 움직여 본 적 있나요?

같은 질문 **"한국의 수도는?"**을 던져도:

| Temperature | AI의 답변 |
|:-----------:|-----------|
| **0** | "서울입니다." (매번 똑같아요) |
| **0.7** | "서울이에요." / "대한민국의 수도는 서울이에요." |
| **2.0** | "부산... 아니 평양? 서울이긴 한데 달에도 수도가..." |

왜 이런 일이 벌어지는 걸까요? 오늘은 AI가 답을 고르는 **비밀 원리**를 파헤쳐 볼 거예요.

---

## 개념 1: AI는 다음 단어의 확률을 계산해요

### 스마트폰 자동완성을 떠올려 봐요

스마트폰에서 "나는 오늘"이라고 치면, 다음에 올 단어를 추천해 주잖아요?

AI도 **완전히 같은 원리**예요. 문장의 앞부분을 보고, 다음에 올 단어의 **확률**을 계산해요.

<div class="key-concept" markdown>

**핵심 개념: AI의 텍스트 생성 원리**

AI는 문장을 한 번에 만들지 않아요. **한 단어씩** 다음에 올 확률이 높은 단어를 골라서 이어 붙여요. 마치 자동완성을 계속 누르는 것처럼요.

</div>

예를 들어, **"나는 오늘 학교에서 ___"** 다음에 올 단어의 확률은 이런 식이에요:

```
공부를  ██████████████████  35%
밥을    ████████████████    30%
친구와  ██████████        15%
잠을    ██████            10%
축구를  ████               5%
노래를  ███                3%
기타    ██                 2%
```

AI는 이 확률을 보고 단어를 **하나 골라요**. 그런데 **어떻게** 고르느냐가 문제예요!

---

## 개념 2: 확률 분포란?

### 주사위로 이해해 봐요

=== "공정한 주사위"

    보통 주사위는 1~6 각 면이 나올 확률이 **똑같아요** (각각 약 16.7%).

    ```
    1  ████  16.7%
    2  ████  16.7%
    3  ████  16.7%
    4  ████  16.7%
    5  ████  16.7%
    6  ████  16.7%
    ```

    어떤 숫자가 나올지 **예측하기 어려워요** — 골고루 나오니까요.

=== "조작된 주사위"

    만약 6이 나올 확률이 엄청 높은 주사위가 있다면?

    ```
    1  █      2%
    2  █      3%
    3  █      3%
    4  ██     7%
    5  ███   10%
    6  ██████████████████  75%
    ```

    이건 거의 항상 **6이 나와요**. 결과를 **예측하기 쉽죠**.

!!! note "확률 분포"
    여러 가지 결과가 나올 수 있을 때, 각 결과가 나올 확률을 모아놓은 것을 **확률 분포**라고 해요. 위의 막대그래프가 바로 확률 분포예요!

이제 이걸 AI에 적용해 봐요. AI가 계산한 "다음 단어의 확률"도 확률 분포예요. 그리고 **Temperature는 이 분포의 모양을 바꿔요!**

## 개념 3: Temperature의 역할

Temperature는 AI가 단어를 고를 때 **확률 분포를 얼마나 고르게 만들지** 결정하는 값이에요.

=== "Temperature = 0 (얼음처럼 차갑게)"

    ```
    공부를  ████████████████████  100%
    밥을                            0%
    친구와                          0%
    잠을                            0%
    ```

    1등 단어만 **무조건 선택**해요. 매번 같은 답이 나와요.

    용도: 정확한 답이 필요할 때 (수학 문제, 코드 작성)

=== "Temperature = 0.7 (적당한 따뜻함)"

    ```
    공부를  ██████████████████  40%
    밥을    ████████████████    30%
    친구와  ████████            15%
    잠을    ██████              10%
    ```

    확률이 높은 단어가 여전히 잘 뽑히지만, 가끔 다른 단어도 선택돼요.

    용도: 일반적인 대화, 글쓰기

=== "Temperature = 2.0 (뜨겁게 달궈!)"

    ```
    공부를  ██████    18%
    밥을    ██████    17%
    친구와  █████     16%
    잠을    █████     16%
    축구를  █████     15%
    노래를  ████      12%
    ```

    거의 **균등해져요**! 어떤 단어든 비슷한 확률로 뽑혀요.

    용도: 창의적인 글, 브레인스토밍 (하지만 엉뚱할 수 있어요)

<div class="key-concept" markdown>

**핵심 비유: 주사위의 면**

- **Temperature가 낮으면** → 조작된 주사위 (1등만 나옴)
- **Temperature가 높으면** → 공정한 주사위 (아무 면이나 나옴)
- Temperature는 주사위의 **면을 얼마나 균등하게** 만들지 결정해요!

</div>

!!! warning "흔한 오해"
    "Temperature를 높이면 AI가 더 똑똑해진다"는 **아니에요!** Temperature를 높이면 AI가 **낮은 확률의 단어도 선택할 수 있게** 될 뿐이에요. 그래서 창의적일 수도 있지만, 엉뚱한 말을 할 수도 있어요.

## 개념 4: 이것이 왜 중요한가

이 원리를 알면, AI를 **목적에 맞게** 쓸 수 있어요!

| 상황 | 추천 Temperature | 이유 |
|------|:-----------------:|------|
| 코딩할 때 | 0 ~ 0.2 | 정확한 코드가 필요하니까 |
| 일반 대화 | 0.5 ~ 0.7 | 자연스럽지만 안정적으로 |
| 시 쓸 때 | 0.8 ~ 1.2 | 창의적인 표현이 필요하니까 |
| 브레인스토밍 | 1.0 ~ 1.5 | 다양한 아이디어가 필요하니까 |
| 실험/장난 | 1.5 ~ 2.0 | 예측 불가능한 재미! |

!!! tip "AI는 '생각'하지 않아요"
    AI가 "나는 서울이 수도라고 생각해"라고 말해도, 실제로는 **확률 계산 → 샘플링**을 반복하고 있을 뿐이에요. AI의 모든 답은 **주사위 굴리기**의 결과예요. 이걸 아는 것과 모르는 것은 AI를 대하는 태도를 완전히 바꿔놓아요.

## 실습 안내

### 실습 1: Temperature 실험실

1. [ChatGPT Playground](https://platform.openai.com/playground)에 접속해요 (또는 선생님이 준비한 환경)
2. 같은 질문을 Temperature **0, 0.7, 1.5**로 각각 3번씩 보내봐요
3. 추천 질문:
    - "한국에서 가장 맛있는 음식은?"
    - "미래의 직업 3가지를 추천해줘"
    - "오늘 날씨를 시로 표현해줘"

!!! note "관찰 포인트"
    - Temperature 0일 때: 매번 같은 답이 나오나요?
    - Temperature 1.5일 때: 어떤 재미있는(또는 이상한) 답이 나왔나요?

### 실습 2: AI 작가 대회

1. 같은 첫 문장으로 시작해요: **"어느 날 학교 뒷산에서 이상한 소리가 들렸다."**
2. **Temperature 0.3**으로 이야기를 만들어봐요 → 안정적이지만 뻔한 이야기
3. **Temperature 1.5**로 이야기를 만들어봐요 → 예측 불가능한 전개!
4. 모둠에서 투표: 어느 쪽이 더 재미있었나요?

### 실습 3: 확률 예측 체험

AI가 다음 단어를 예측하는 과정을 직접 체험해 봐요!

- 문장의 빈칸에 들어갈 단어를 **확률 순서대로** 예측해 봐요
- AI의 예측 결과와 비교해 봐요

## 자주 하는 오해

??? success "오해 1: Temperature를 높이면 AI가 더 창의적으로 '사고'한다?"
    **아니에요!** AI는 사고하지 않아요. Temperature를 높이면 **낮은 확률의 단어도 선택될 수 있게** 만들 뿐이에요. 창의적으로 보이는 건 **우연히 색다른 조합**이 나왔기 때문이에요.

??? success "오해 2: Temperature가 높을수록 좋다?"
    **상황에 따라 달라요!** 코드를 짜달라고 했는데 Temperature가 2.0이면, 변수 이름에 엉뚱한 단어가 들어가서 작동하지 않는 코드가 나올 수 있어요.

??? success "오해 3: AI가 '서울'이라고 답하면 서울을 알고 있는 거다?"
    AI는 "한국의 수도는"이라는 패턴 다음에 "서울"이 올 확률이 높다는 것을 **통계적으로** 학습한 거예요. "안다"와 "확률이 높다"는 다른 의미예요.

## 정리

| 핵심 용어 | 의미 | 비유 |
|-----------|------|------|
| **확률 분포** | 각 단어가 선택될 확률을 모아놓은 것 | 주사위 각 면의 확률 |
| **샘플링** | 확률 분포에서 단어를 하나 고르는 과정 | 주사위를 굴리는 것 |
| **Temperature** | 확률 분포를 균등하게/뾰족하게 만드는 값 | 주사위 면의 균등함을 조절하는 다이얼 |
| **Temperature 0** | 항상 확률 1등 단어만 선택 | 한 면만 있는 주사위 |
| **Temperature 높음** | 모든 단어가 비슷한 확률로 선택 | 공정한 주사위 |

<div class="key-concept" markdown>

**오늘의 핵심 한 줄**

AI의 모든 답은 주사위 굴리기. Temperature는 주사위의 면 수를 바꾼다.

</div>
