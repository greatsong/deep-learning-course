# 2차시 교사용 지도서: 확률과 Temperature

## 수업 목표

- AI가 다음 단어를 **확률적으로** 선택한다는 것을 이해한다.
- Temperature가 확률 분포에 미치는 영향을 체험한다.
- 용도에 따라 Temperature를 조절하는 이유를 설명할 수 있다.

## 도입 시나리오 (0~5분)

### 교사 대사

> "ChatGPT Playground를 열어볼게요."

(화면 공유, Playground 열기)

> "'한국의 수도는?'이라고 물어볼게요. Temperature를 0으로 설정..."

(입력, 결과 확인)

> "답이 뭐죠? '서울'. 다시 물어볼게요."

(같은 질문 재입력)

> "또 '서울'. 한 번 더?"

(같은 질문 재입력)

> "'서울'. 이제 Temperature를 2.0으로 올려볼게요. 같은 질문..."

(결과 읽기 — 높은 Temperature에서 나오는 엉뚱한 답을 읽어준다)

> "(결과 읽기) 뭐라는 거죠?"

(학생들 웃음)

> "같은 질문인데 왜 답이 달라질까요? 이 슬라이더는 뭘 하는 걸까요?"

!!! tip "시연 팁"
    Temperature 2.0의 결과가 항상 웃기지는 않습니다.
    수업 전에 미리 여러 번 테스트해서 재미있는 결과가 나오는 질문을 찾아두세요.
    대안 질문: "봄이 오면", "학교에서 가장 좋아하는 건"

|-----------|-------------|
| "Temperature가 높으면 AI가 미쳤나요?" | "미친 게 아니라 **주사위의 면이 균등해진** 거예요. 꼴찌 단어도 선택될 수 있으니까 엉뚱한 답이 나오는 거죠." |

| "AI가 생각해서 답하는 거 아니에요?" | "아니에요! **확률 계산 후 주사위를 굴리는** 거예요. 이게 오늘의 핵심이에요." |

| "Playground는 뭐예요?" | "ChatGPT의 '실험실' 같은 거예요. 일반 ChatGPT보다 세부 설정을 조절할 수 있어요." |

## 이론 설명 (5~15분)

### 1단계: AI의 다음 단어 선택 (4분)

> "지난 시간에 AI가 글자를 토큰으로 쪼갠다고 배웠죠? 토큰을 쪼갠 다음에는 뭘 할까요?"

(잠시 멈춤)

> "다음 토큰을 **예측**해요. 어떻게? 모든 가능한 다음 단어에 **확률**을 매겨요."

**칠판/슬라이드 예시:**

```
"나는 오늘 학교에 ___"

갔다    → 45%
왔다    → 25%
간다    → 15%
안 갔다 → 10%
날았다  →  3%
춤췄다  →  2%
```

> "이 확률표에서 하나를 골라요. 매번 확률에 따라 주사위를 굴리니까, 매번 다른 답이 나올 수 있는 거예요."

### 2단계: Temperature의 역할 (4분)

> "그런데 이 확률표를 조절할 수 있어요. 그게 Temperature예요."

**비유: 주사위**

> "보통 주사위를 생각해보세요. 6면이 균등하죠. 이게 Temperature가 높은 상태예요. 모든 단어가 비슷한 확률로 선택돼요."

> "이제 주사위에 자석을 붙여서 1번 면이 거의 항상 나오게 만들었다고 생각해보세요. 이게 Temperature가 낮은 상태예요. 확률이 가장 높은 단어가 거의 항상 선택돼요."

**슬라이드에 시각적 비교:**

```
Temperature = 0 (얼음처럼 차가운)
갔다 ████████████████████ 99%
왔다 █                     1%
간다                       0%

Temperature = 1.0 (기본값)
갔다 █████████             45%
왔다 █████                 25%
간다 ███                   15%
안 갔다 ██                 10%
날았다 █                    3%
춤췄다                      2%

Temperature = 2.0 (뜨겁게 달아오른)
갔다 ████                  20%
왔다 ████                  18%
간다 ███                   17%
안 갔다 ███                16%
날았다 ███                 15%
춤췄다 ███                 14%
```

> "Temperature가 높아질수록 확률이 균등해져요. 그래서 '날았다', '춤췄다' 같은 엉뚱한 단어도 선택될 수 있는 거예요."

### 3단계: 왜 Temperature라고 부를까? (2분)

> "물리학에서 온도가 높으면 분자들이 활발하게 움직이잖아요? 비슷해요. Temperature가 높으면 단어 선택이 활발해지고(다양해지고), 낮으면 조용해져요(한 가지로 수렴해요)."

!!! note "교사 참고"
    Temperature라는 이름이 통계역학의 볼츠만 분포에서 왔다는 것은 상위 학생에게만 간단히 언급합니다. "0차시에서 노벨 물리학상 얘기 했죠? 바로 그 연결이에요."

## 실습: AI 작가 대회 (15~40분)

### 실습 개요

!!! success "활동 목표"
    같은 첫 문장으로 Temperature를 바꿔가며 글을 생성하고, 결과를 비교합니다.

### 실습 단계

#### 1단계: 설정 (5분)

- ChatGPT Playground 접속 (교사 계정으로 화면 공유하거나, 학생 계정 사용)
- 첫 문장 통일: **"어느 날 학교에서"**

!!! warning "Playground 접속 관련"
    학생들이 개인 OpenAI 계정이 없을 수 있습니다.
    **방법 1:** 교사가 화면 공유하며 학생들이 돌아가며 요청
    **방법 2:** 모둠별로 1개 계정 공유
    **방법 3:** 교사가 미리 4개 결과를 생성해두고, 학생들은 비교/분석에 집중

#### 2단계: 4가지 Temperature로 생성 (10분)

| Temperature | 기대되는 결과 |
|-------------|---------------|
| 0.3 | 평범하고 예측 가능한 이야기 |
| 0.7 | 자연스럽지만 약간의 변화가 있는 이야기 |
| 1.0 | 창의적이고 다양한 표현의 이야기 |
| 1.5 | 예상치 못한 전개, 가끔 문맥이 어긋남 |

**각 결과를 활동지에 기록하게 합니다.**

#### 3단계: 비교 & 투표 (7분)

- 4개 결과를 모둠 내에서 소리 내어 읽기
- **가장 재미있는 글**에 투표
- **가장 정확한 글**에 투표
- "재미있는 글"과 "정확한 글"이 다른 Temperature인지 확인

#### 4단계: 발표 & 토론 (3분)

- 각 모둠의 투표 결과 공유
- 교사 정리:

> "재미있는 글은 Temperature가 높을 때, 정확한 글은 낮을 때 나오는 경향이 있죠? Temperature를 조절하는 것은 **AI의 창의성을 조절하는 것**이에요."

### 추가 활동 (시간 여유 시)

**용도별 최적 Temperature 찾기:**

| 용도 | 학생이 찾을 Temperature |
|------|------------------------|
| 수학 문제 풀이 | 낮은 값 (0~0.3) |
| 시 쓰기 | 높은 값 (0.8~1.2) |
| 번역 | 낮은 값 (0~0.3) |
| 브레인스토밍 | 높은 값 (1.0~1.5) |
| 코드 작성 | 낮은 값 (0~0.5) |

> "이제 여러분은 ChatGPT를 쓸 때 Temperature를 조절할 수 있어요. 정확한 답이 필요하면 낮추고, 창의적인 답이 필요하면 올리면 돼요."

## 실습 중 순회 지도 포인트

### 상위 학생

- **소프트맥스 함수** 개념 간단히 소개: "확률을 만드는 수학 함수가 있어요"
- Temperature가 소프트맥스 함수의 어디에 들어가는지 직관적으로 설명
- "Temperature 0이 정확히 뭘 의미하는지" 질문 유도 (argmax, greedy decoding)

### 중위 학생

- 활동지 순서대로 진행하는지 확인
- "Temperature 0.7이 ChatGPT의 기본값이에요"라는 정보 제공
- 비교 결과를 자기 말로 정리하게 하기

### 하위 학생

!!! warning "지원 방법"
    - 교사가 함께 Playground를 조작
    - "이 슬라이더를 이쪽으로 밀어볼까요? 결과가 어떻게 달라지는지 봐요"
    - 4개 결과를 미리 인쇄해두고 **비교하는 활동**에만 집중시키기
    - "주사위 비유"를 반복 활용: "Temperature 높으면 공정한 주사위, 낮으면 조작된 주사위"

## 자주 나오는 질문

!!! question "Temperature를 0으로 하면 매번 똑같은 답이 나오나요?"
    "거의 같아요. 확률이 가장 높은 단어만 계속 고르니까요. 아주 미세한 차이가 있을 수 있지만, 사실상 같은 답이 나옵니다."

!!! question "Temperature 말고 다른 설정도 있어요?"
    "네! Top-p, Top-k 같은 설정도 있어요. 이건 '후보 단어의 범위를 제한하는' 방식이에요. 오늘은 Temperature만 이해하면 충분해요."

!!! question "ChatGPT 기본 Temperature는 뭐예요?"
    "대략 0.7 정도예요. 너무 딱딱하지도, 너무 엉뚱하지도 않은 균형점이에요."

!!! question "AI가 정말 주사위를 굴려요?"
    "비유예요. 실제로는 난수 생성기(random number generator)를 써요. 컴퓨터가 무작위 수를 만들어서 확률에 따라 단어를 고르는 거죠."

!!! question "Temperature를 10으로 하면 어떻게 돼요?"
    "확률이 거의 완전히 균등해져서 사실상 랜덤 단어가 나와요. 의미 없는 문장이 되죠. 그래서 보통 0~2 사이에서 조절해요."

## 정리 (40~50분)

### 핵심 정리 (5분)

> **오늘 배운 것:**
>
> 1. AI는 다음 단어를 **확률**로 고른다 (생각하는 게 아니다!)
> 2. **Temperature**는 확률 분포의 균등함을 조절하는 슬라이더
> 3. 낮은 Temperature = 정확하고 예측 가능 / 높은 Temperature = 창의적이고 다양

### 1분 퀴즈

> "코드를 짤 때 Temperature를 어떻게 설정하면 좋을까요?"
> (학생: "낮게요!")
> "시를 쓸 때는?"
> (학생: "높게요!")
> "완벽해요."

### 다음 차시 예고

> "자, 이제 AI가 토큰을 쪼개고, 확률로 다음 단어를 고르는 건 알겠어요. 그런데 한 가지 빠진 게 있어요. AI가 '왕 - 남자 + 여자 = 여왕'이라는 계산을 할 수 있대요. 단어로 덧셈 뺄셈을? 다음 시간에 그 비밀을 알아봅니다."
